Iconify SVG framework makes using icons as easy as icon fonts. To use "facebook" in HTML, add this code to the document:

<span class="iconify" data-icon="mdi:facebook"></span>

Iconify SVG framework will load icon data from Iconify API and replace that placeholder with SVG.

Make sure you import Iconify SVG framework:

<script src="https://code.iconify.design/2/2.2.1/iconify.min.js"></script>
